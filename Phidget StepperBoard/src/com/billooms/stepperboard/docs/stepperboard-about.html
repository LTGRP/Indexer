About Phidget StepperBoard

The status of the connected Phidgets 1062/1063 StepperBoard can be monitored by selecting the "Window" -> "StepperBoard" menu item. This will display a window on the right side of the display area which will remain open until explicitly closed by the user. This window is only for monitoring the state of the board, and data cannot be entered/modified using this window. This window is particularly useful for debugging purposes in the event of unexpected behavior of the stepper motor.
