About IndexWheels

A software-based IndexWheel is essentially the same as a regular lathe index wheel which has been made out of metal with holes in it. It is defined by the number of holes (which the user can specify) which are equally spaced around the wheel. Some of the holes can be "filled" which means that the IndexWheel will stop there during rotation. Other holes can be left "un-filled" which means that the IndexWheel will not stop there during rotation.

In addition to having some number of holes, the IndexWheel can be phase-shifted or offset by some amount. The amount is specified as being anywhere within the range of one hole spacing by setting a phase shift number in the range of 0.0 to 1.0.

Any number of IndexWheels can be described. The user can specify a unique name for each wheel for convenience. All IndexWheels are rotated together as if one had multiple metal plates on the spindle of a lathe that rotated together.
