About Indexer XML Files

Information is saved in files using an .xml file format. This is a somewhat human-readable format that can be edited by any text editor with care. A document type definition (dtd) for the file is available at www.billooms.com/dtds/indexer1_0.dtd. The second line in the xml files contains a link to this file. By default, it is commented out to avoid problems if you are using a computer that is not connected to the internet. If you want to edit the file manually, un-comment the second line and use an editor that understands xml formats. Your editor will have a command to validate that the edits you made conform to the rules in the dtd file.

Menu Actions

File

  • New File... -- Create a new file from a default template and open it.
  • Open XML File... -- Open a previously saved xml file
  • Save XML File -- Save the file after edits have been made.
  • Save XML File As... -- Save a copy of the data to a new file, leaving the old file unchanged.
  • Revert to Saved File -- Go back to the last saved version of the file.
  • Save All -- After a file has been opened and modified, this action becomes available as a reminder that changes have been made.
